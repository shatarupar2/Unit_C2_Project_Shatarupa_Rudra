Formatting Tags For HTML

Styling Tags

After learning the essential tags to make a HTML document work. Listed below are the styling tags to better format the document.

  1. <b> : makes the text appear bold
  2. <strong> : also makes the text bold
  3. <i> : makes the text italics
  4. <em> : also makes the text italic
  5. <mark> : makes the text highlighted
  6. <small> : makes the text smaller
  7. <del> : makes the text crossed out
  8. <ins> : makes the text underlined
  9. <sub> : makes the text subscripted
  10. <sup> : makes the text superscripted
  11. <ol> : creates an ordered list
  12. <ul> : creates a unordered list
  13. <li> : is used for the items within the lists

The code setup whe making a list should either be:

      <ol>
        <li> <li>
        <li> <li>
        <li> <li>
        ...as many li tags for as many things needed to add. 
      </ol>

to make a ordered list

OR

      <ul>
        <li> <li>
        <li> <li>
        <li> <li>
        ...as many li tags for as many things needed to add. 
      </ul>

to make a unordered list

Heading And Text Break Tags

  1. < h1> : the top-level heading
  2. < h2> : the second level heading
  3. < h3> : the third level heading
  4. < h4> : the fourth level heading
  5. < h5> : the fifth level heading
  6. < h6> : the sixth level heading
  7. <p> : creates a paragraph on the webpage
  8. <br> : creates a line break
  9. <hr> : creates a line between blocks of text
  10. <pre> : preserves the formatting of the text
