Tables

Why Do Developers Use Tables?

Just like in day-to-day life, tables allow developers to organize information with structure similar to a grid.

Essential Elements for Tables

  1. <table> : creates the table
  2. <tr> : creates a row in the table
  3. <th> : creates a heading cell in the table
  4. <td> : creates a data cell in the table

Structure for Code

        <table>
            <tr>
              <th>Heading 1</th>
              <th>Heading 2</th>
            </tr>
            <tr>
              <td>Data 1</td>
              <td>Data 2</td>
            </tr>
          </table>

How to Break a Cell for More than One Row or Column?

To make a row fit into 2 cells, use the code:
      <th rowspan="2">
    
 To make a column fit into 2 cells, use the code:
      <th colspan"2">
