Head Elements and Entities

What are Head Elements?

Head Elements are used to state any information about the metadata of the HTML document without actually being shown on the website.

What should go in the head section?

  • Title - which the title of the HTML document that shows up at the top of the web page window.
  • Metadata - any information that helps web crawlers identify and organize the HTML document, not visible to the web users.
  • Styles - are the instruction for how the website should look.
  • Scripts - are the JavaScript codes that run before the web page begins.
  • Links - any inportant links of files that are connected to the web pages.

Examples of Tags that Go Into the Head

  • <title>I LOVE CODE<title>
  • <meta charset = "UTF-8">
  • <meta name="author" content="Author">
  • <meta name="description" content="This is a description of my awesome website.">
  • <link rel="stylesheet" href="styles.css">
  • <script src="script.js"></script>
  • <html lang="en-US">
  • <meta name="viewport" content="width=device-width, initial-scale=1.0">
  • <link rel="icon" href="favicon.ico" type="image/x-icon">

Entities

An entity refers to a specific character or symbol represented by a unique code point.

Unicode

Unicode is a character encoding standard that assigns a specific code point to every character used in written languages, symbols, and emojis. It is used to provide a universal system for representing text in code.
