Image Insertion, Link Insertion, & Iframes

Image Insertion

How to Insert an Image onto a HTML document?

To insert local images within your computer, you could simply use this code:
      <img src="*image file name*.jpg (or.png)" alt="*description of image being shown*"> style="width:*preferred size*px" 
    
    An example of this would be: 
     <img src="moonpic.jpg" alt"A pic of a Moon" style="width:400px"> 
    
To insert external, remote images, you must copy the link address into a format like this:
      <img src="*insert link address*" alt="*insert description of image*">

    An example of this would be: 
      
        
          The code for this was: 
            <img src="https://images-assets.nasa.gov/image/PIA00405/PIA00405~large.jpg?w=1920&h=1920&fit=clip&crop=faces%2Cfocalpoint" alt="An Pic of the Moon" style="width:400px">

Link Insertion

How to Insert a Link onto a HTML document?

To link sites coming from an external source, the code used is this:
       <a href="*insert website link using https"> *insert website name* </a>

    An example of this would be:
       The Google Classroom Website
            
       The code for this was:
            <a href="https://classroom.google.com/u/0/h">The Google Classroom Website</a>
    
However, to link something that is internal, the code setup used is this: 
        <a href="*pathToFile starting  ./ or ../*">*File Name*</a>
       
    An example of this would be:
       Home Page of this HTML Document
            
       The code for this was:
            <a href="./index.html">Home Page of this HTML Document</a>

Iframes

What are Iframes used for?

Developers use iframes as a way to display the content of another website on an HTML document.

How to insert Iframes?

To use iframes in an HTML document, use the code: 
        <iframe src="url" title="description"></iframe>

    An example of this would be: 
    

          The code for this was:
           <iframe src="https://2048.gg/cupcakes" title="2048 - Cupcake Version" width="200" height="200"></iframe>
