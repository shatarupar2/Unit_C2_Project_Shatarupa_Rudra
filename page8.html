Semantic Elements

What are Semantic Elements

Semantic elements are HTML elements that describe their meaning and purpose. These elements improve organization, readability, and accessibility for all users.

Examples of Semantic Tags

Elements Meaning Location on Web Page
<header> Signifies the start of the page Top of the page
<footer> Signifies the end of the page End of the page
<main> Contains the main content of the page Within the body of the page
<nav> Allows users to navigate links of/to other sites Within the body, usually before a list
<aside> Contains any information indirectly related to the main content Within the body, usually after main content
<article> Contains any information separate from anything else on the page Within the body, in the main tag
<section> Groups content into a specific location Within the body, in the main, footer, and header

Why are semantic elements important?

Using semantic tags makes the HTML document more accessible to people with disabilities, organizes the code better so it's more readable. This fits with W3C standards for inclusion.
