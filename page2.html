Basic Start of HTML Pages

What is HTML?

HTML is a coding language for the World Wide Web that stands for HyperText Markup Language. HTML is used to format the text using tags, like <p>.

Must Needed Tags for HTML

  1. Every webpage should begin with <!DOCTYPE html>. This identiies the page as HTML verison 5, shows the start of the web page. THIS IS IMPORTANT AS IT TELLS THE BROWSER WHAT TYPE OF DOCUMENT TO EXPECT.
  2. After the !DOCTYPE tag, the following code should always be <html>. The html tags states the roots for the HTML site. The lang="en" should also follow behind the html tag as it states the used language.
  3. After <html>, the head tag is needed (<head>), and it indicates the head section of the html document.
  4. In the head tag, goes <title>. The title tag gives the words that show up on web browser tags.
  5. After, a body tag should be placed (<body>) This contains all the the visible content of the page.
 The basic code for an html document should look like this 
    
    
    <!DOCTYPE html>
    <html> 
    <head>
    <title>HTML Tutorial</title>
    </head>
    <body>


    
    </body>
    </html>
